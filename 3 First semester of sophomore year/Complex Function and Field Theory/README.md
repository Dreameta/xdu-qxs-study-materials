# 2 教材

总体感觉，这门课学的还是较少，考的较深。

1. 书本的电子版主要用来布置作业，西交那版。
2. 推荐一本 《复变函数方法论》这本书大致能看到复变的一些基础知识逻辑和结构。可以感觉到工科本科真的触及不到数学秃头的“乐趣"，我们学的复变也只是其中的一小部分，对函数本身理解。和学数学的同学交流过，感觉数学真的好难，曾经幻想学点数学，后来发现没有时间，也没有天赋（逃）。
